# Memory game

## A Flutter Flame application to play the memory game.

There are 3 levels : 
- easy : 6 tiles
- medium : 16 tiles
- hard : 30 tiles

You have to tap on two tiles that have the same color/id to reveal them. The game ends when you have revealed everything.

## The application is available [here](https://GrosGabriel.github.io/memory-game/)




## Running the app locally : 

Run the command `flutter pub get` to install the dependencies.

Then run `flutter run -d web-server --web-port=5000` in the root folder.
